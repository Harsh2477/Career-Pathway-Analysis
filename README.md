# Career-Pathway-Analysis

# About
This project analyzes career trends, skills demand, salary patterns, and job market opportunities across different industries. The goal is to identify factors that influence career growth and help individuals make informed career decisions using data-driven insights.

# Objective
The objective was to analyse a dataset containing the educational backgrounds and career progression of Data Scientists and Machine Learning Engineers — and answer one core question: what is the most common and successful path someone takes to land a job as a Data Scientist or ML Engineer

# Tools
Pandas, Numpy, Scikit-learn , matplotlib and seaborn.
